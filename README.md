# FB Auto Unfollow

Automate bulk unfollowing of people, pages, and groups on Facebook safely.

---

## Description
This JavaScript snippet allows you to unfollow multiple people, pages, and groups on Facebook with an adjustable delay between actions. It is designed to reduce the risk of Facebook detecting automated behavior.

> ⚠️ **Warning:** Use at your own risk. Automating actions may violate Facebook’s terms of service. Only run on your own account.

---

## Features
- Unfollow multiple users/groups at once
- Adjustable delay between clicks
- Logs progress in the browser console
- Optional auto-scrolling (if implemented)

---

## Usage
1. Open Facebook on a desktop browser.
2. Go to **News Feed → Edit Preferences → Unfollow people/groups**.
3. Open the browser console (F12 → Console tab).
4. Paste the contents of `unfollow.js` into the console and press **Enter**.
5. Watch the console logs for progress:
